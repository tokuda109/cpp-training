# C++ Training

## Dependencies

* [CMake](https://cmake.org) (v3.8.2)
* [OpenCV](http://opencv.org) (v3.2.0)
* [Boost](http://www.boost.org) (v1.64.0)
* [Google Test](https://github.com/google/googletest) (v1.8.0)

```sh
$ brew install cmake
$ brew install boost
$ brew install opencv3 --with-contrib
$ cmake .
$ make
```
